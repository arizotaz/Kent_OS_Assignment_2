# Banker's Algorithm Deadlock Avoidance

This project implements the Banker's Algorithm to determine whether a system is in a safe state and to output the safe sequence if one exists.

## Files
- `banker.cpp` — Main program implementing the Banker's Algorithm  
- `Makefile` — Build instructions  
- `input.txt` — Input data file containing Allocation, Max, and Available matrices  
- `README.md` — This file  
- `example.txt` — Showcase of execution and output
